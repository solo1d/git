# \*重命名远程版本库的分支

```bash
远程操作: 重命名远程版本库的分支
    首先必须先把远程版本库拉取到本地. 然后删除远程版本库的某个分支
          $git pull        # 拉取最新的远程版本库到本地
   
     然后进入到要重命名的分支
          
          $git branch -m +旧分支名字  +新分支名字
          
          $git push origin --delete  +重命名的分支名字
          
          $git push --set-upstream origin +重命名完成的分支名字    
      这样就完成推送并且重命名了.
```
